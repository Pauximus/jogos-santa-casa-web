# V68 — Notificações Inteligentes

Esta versão transforma o Push Engine numa base automática para notificações úteis.

## Inclui

- Modo `test` para teste manual.
- Modo `reminder` para lembrete de sorteio.
- Modo `results` para avisar que os resultados já podem ser analisados.
- Modo `scheduled` para funcionamento automático por horário.
- Proteção contra notificações repetidas através da tabela `notification_log`.
- Compatibilidade com GitHub Actions e Node 24.

## Próxima fase

A próxima evolução será ligar o motor aos resultados oficiais e calcular prémios automaticamente.
